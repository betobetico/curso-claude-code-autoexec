## Concepto Clave del Curso: Context Engineering

### Del Prompt Engineering al Context Engineering

```
┌─────────────────────────────────────────────────────────────────┐
│  ANTES (ChatGPT, etc.)         AHORA (Claude Code)             │
│  ════════════════════          ═══════════════════             │
│                                                                 │
│  Solo texto                 →  Texto + Ficheros + Herramientas │
│  "Escribe mejor el prompt"  →  "Dame el contexto completo"     │
│  Copiar/pegar contenido     →  Acceso directo a tus archivos   │
│  Limitado a una ventana     →  Tu proyecto entero como contexto│
└─────────────────────────────────────────────────────────────────┘
```

**Los ficheros son el nuevo prompt.**

## Microlección 1: Bienvenida

### Concepto Principal
Primera interacción con Claude Code usando lenguaje natural.

### Lo que aprende el estudiante
- Claude Code entiende **lenguaje natural** - no necesita sintaxis especial
- Puede pedir cosas como "Dame un resumen de qué archivos hay"
- La diferencia entre comandos técnicos (`ls -la | grep`) y peticiones en español
- El terminal es un espacio de conversación, no solo de comandos

### Habilidades practicadas
- Ejecutar su primer comando en lenguaje natural
- Hacer preguntas exploratorias sobre archivos
- Entender la respuesta de Claude

---

## Microlección 2: Exploración

### Concepto Principal
Navegar y entender estructuras de archivos antes de actuar.

### Lo que aprende el estudiante
- Cómo obtener un **panorama completo** de una carpeta
- Identificar categorías y patrones en nombres de archivos
- Ver archivos de forma visual (tree, listados)
- Analizar por tipo, por cliente, por fecha
- La importancia de **entender antes de actuar**

### Habilidades practicadas
- Preguntas como "¿Cuántos archivos hay y de qué tipos?"
- Identificar clientes en nombres de archivos
- Visualizar estructura con vistas de árbol
- Proponer criterios de organización propios

---

## Microlección 3: Organización (WOW Moment)

### Concepto Principal
Automatización masiva - organizar 86 archivos en segundos.

### Lo que aprende el estudiante
- Claude puede **mover archivos automáticamente** según criterios
- La diferencia entre organización manual (horas) y automática (segundos)
- Estructuras de carpetas (cliente > tipo vs tipo > cliente)
- El poder transformador de la automatización

### Habilidades practicadas
- Proponer una estructura de organización
- Ejecutar scripts de organización masiva
- Verificar resultados en Finder
- Reflexionar sobre arquitectura de información

---

## Microlección 4: Extracción de Datos

### Concepto Principal
Leer y extraer información de múltiples archivos sin abrirlos.

### Lo que aprende el estudiante
- Claude puede **leer contenido** de documentos
- Extraer datos específicos (montos, fechas, nombres)
- Crear tablas y resúmenes consolidados
- Procesar múltiples archivos a la vez

### Habilidades practicadas
- Ver contenido de un archivo específico
- Extraer totales de todas las facturas
- Crear tablas con cliente, archivo y monto
- Generar informes ejecutivos automáticos

---

## Microlección 5: Análisis y Síntesis

### Concepto Principal
Convertir datos en insights accionables.

### Lo que aprende el estudiante
- Los datos por sí solos no son útiles - los **insights** sí
- Cómo detectar patrones (volumen, actividad, fase comercial)
- Generar recomendaciones basadas en datos, no en intuición
- Inteligencia de negocio sin herramientas caras

### Habilidades practicadas
- Análisis de volumen por cliente
- Identificar clientes activos vs inactivos
- Detectar fase comercial (prospecto, onboarding, recurrente)
- Generar recomendaciones automáticas de acción

---

## Microlección 6: Generación de Contenido

### Concepto Principal
Crear documentos profesionales automáticamente.

### Lo que aprende el estudiante
- Claude puede **generar documentos** completos
- Índices navegables que se actualizan
- README profesionales con estructura clara
- Informes listos para enviar a clientes o superiores

### Habilidades practicadas
- Crear índice de todos los documentos
- Generar README con estructura del archivo
- Producir informes ejecutivos con métricas
- Personalizar templates según necesidades

---

## Microlección 7: Investigación Web

### Concepto Principal
Enriquecer datos internos con información externa.

### Lo que aprende el estudiante
- Claude puede **buscar en la web** (con WebSearch)
- Crear fichas de cliente enriquecidas
- Combinar información interna y externa
- Limitaciones: privacidad, fuentes, verificación

### Habilidades practicadas
- Crear fichas de cliente con datos de empresa
- Identificar ejecutivos clave
- Detectar oportunidades de negocio
- Preparar información para reuniones

---

## Microlección 8: Proyecto Final

### Concepto Principal
Aplicar todo lo aprendido de forma autónoma.

### Lo que aprende el estudiante
- **Dirigir** a Claude en vez de seguir instrucciones
- Tomar decisiones propias sobre organización
- Crear un sistema completo de gestión documental
- El poder está en saber QUÉ pedir, no en memorizar comandos

### Habilidades demostradas
- Explorar una carpeta nueva sin guía
- Decidir y crear estructura de organización
- Analizar el estado de cada cliente
- Documentar el sistema creado
