# Prompt Inicial del Proyecto

## Origen
Este proyecto nace de la necesidad de crear un curso interactivo de Claude Code que se ejecute completamente dentro del terminal, adaptado al nivel del estudiante.

## Contexto
- **Fecha de inicio:** 2026-01-28
- **Autor:** Alberto Benbunan
- **Inspiración:** ccforeveryone.com pero más dinámico y orientado a resultados

## Prompt Original
> "Basado en el spec.md vamos a desarrollar este curso de Claude Code, vamos a estar claros de 3 cosas:
> 1. Para quien es?
> 2. Cual es el caso que vamos a plantear?
> 3. Como hacemos una primera versión sencilla sin que se pierda la ambición futura de las spec?"

## Decisiones Tomadas

### 1. Audiencia
**Curioso general** con énfasis en consultores, freelancers y emprendedores no-técnicos.

### 2. Escenario
**"El Consultor Abrumado"** - Un consultor independiente con 5 clientes activos necesita organizar el caos documental de su negocio.

### 3. MVP vs. Visión Completa

**MVP (4-6 semanas):**
- Módulo 1 completo (8 lecciones)
- Progreso local
- Slash commands básicos
- Sin backend

**Visión futura (spec.md):**
- WhatsApp con spaced repetition
- Discord y comunidad
- Cohorts con office hours
- Peer review
- Monetización premium

## Objetivo del Negocio
1. Lead magnet para servicios de consultoría
2. Construcción de autoridad en Claude Code (español)
